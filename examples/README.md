# Anonymous examples

These examples use fictional names, values, and content. They demonstrate the
delivery chain without publishing real projects or presenting synthetic images
as client work.

| Surface | Example | Runnable artifact |
|---|---|---|
| Web | [Northstar operations dashboard](web-dashboard/) | `index.html` |
| Desktop | [Quiet settings panel](desktop-settings/) | `demo.py` |
| Game | [Archive save menu](game-save-menu/) | `screen.rpy` |

Each directory includes an approved design lock, a resolved asset manifest,
implementation code, and a comparison summary. The Web example is responsive;
the desktop and game mockups are lightweight SVG references intended for
documentation and automated link checks.
